# translation-ldq

## 一个简单实用的翻译工具

### v1.5.2
1. 实现中英文单词句子的翻译
2. 实现显示格式的优化
3. 实现友好的使用信息提示
4. 实现添加 ```--color``` 参数有彩色输出效果

#### 使用方法：

- 在命令行中输入：

```
$ npm install translation-ldq -g
```
安装本工具

- 输入命令：

```
translation-ldq|tlq translate <word[sentence]> [--color]
```
或
```
translate <word[sentence]> [--color]
```
再回车可查看翻译结果

- 查看帮助:

```
translation-ldq|tlq
```
或
```
translate --help|-h
```
再回车可查看帮助结果
